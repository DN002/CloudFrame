------------------------------------------------
CLOUDFRAME
------------------------------------------------

A Minecraft machinery server plugin for 1.21.XX...

CloudFrame adds multiple machines, tools, etc. to enhance and adds more stuff to explore.
Here are some machines we are currently working on...
- Quarry machine
- Electric smelter
- Power network
- Wind Turbines
- Solar Panels
- And more to come...

** very early stages, most parts are a work in progress **
