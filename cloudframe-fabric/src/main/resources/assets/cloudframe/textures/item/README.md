# CloudFrame Custom Textures

Place your custom item texture PNG files in this directory:

- `marker.png` - Marker wand texture (16x16 PNG)
- `wrench.png` - Wrench texture (16x16 PNG)  
- `silk_touch_augment.png` - Silk Touch augment texture (16x16 PNG)
- `speed_augment_1.png` - Speed I augment texture (16x16 PNG)
- `speed_augment_2.png` - Speed II augment texture (16x16 PNG)
- `speed_augment_3.png` - Speed III augment texture (16x16 PNG)
- `tube.png` - Tube texture (16x16 PNG)
- `quarry_controller.png` - Quarry controller texture (16x16 PNG)

These textures will be automatically loaded by Minecraft when the mod starts.

For now, placeholder textures from vanilla Minecraft are being used.
